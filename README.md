# Examen Blanc – Design Patterns & AOP

## Description
Application Java permettant la manipulation de dessins composés de figures géométriques.
Le projet met en œuvre plusieurs Design Patterns (Composite, Observer, Strategy) ainsi que
la Programmation Orientée Aspect (Spring AOP) pour la journalisation et la sécurité.

## Design Patterns utilisés
- Composite : gestion des groupes de figures
- Observer : notification du changement de paramétrage
- Strategy : algorithmes de traitement interchangeables
- AOP : Logging et Sécurité

## Technologies
- Java 11
- Maven
- Spring AOP
- AspectJ
